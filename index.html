NodeJS - Desarrollo Web

¿Qué es NodeJS?

NodeJS permite usar JavaScript en el backend, o sea, fuera del navegador. Se puede usar para hacer APIs, servidores, manejar bases de datos, etc.

Diferencia entre JS en el navegador y en el backend

  • Frontend: Para botones, animaciones, formularios.
  • Backend: Para manejar datos, peticiones, bases de datos.

Ventajas y desventajas

Ventajas:

  • Usás el mismo lenguaje en cliente y servidor.
  • Mucha comunidad y recursos.
  • Ideal para apps en tiempo real.

Desventajas:

  • No es bueno para tareas pesadas.
  • Puede ser complicado manejar errores si no sabés bien.

Instalación de NodeJS

  1. Entrás a nodejs.org
  2. Descargás la versión recomendada.
  3. Instalás haciendo clic en "Next" hasta terminar.
  4. Verificás en consola:
node -v
npm -v

npm y módulos

Importar y exportar módulos

Archivo operaciones.js:

function sumar(a, b) {
  return a + b;
}

module.exports = sumar;

Archivo index.js:

const sumar = require('./operaciones');
console.log(sumar(2, 3));

¿Qué es npm?

Es el gestor de paquetes de Node. Sirve para instalar librerías externas. Lo mantiene GitHub.

¿Yarn?

Es otra herramienta similar a npm. Tiene buena performance y mejor cacheo.

yarn.lock vs package-lock.json

  • Ambos fijan versiones de paquetes.
  • Uno lo genera yarn, el otro npm.

5 comandos npm útiles

  • npm init – crea package.json
  • npm install – instala paquetes
  • npm uninstall – elimina paquetes
  • npm update – actualiza lo instalado
  • npm list – muestra lo que hay instalado

Flags comunes

  • -g: global
  • --save: ya no se usa mucho
  • -D: dependencia de desarrollo

¿Cómo se usa un módulo?

npm install axios
const axios = require('axios');

¿Y si solo lo quiero para desarrollo?

npm install nodemon -D

Con eso, npm lo guarda en devDependencies.
